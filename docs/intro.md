# Vibe Coding Harness

**Spec Kit**과 **Superpowers 스킬**을 결합하여 AI 기반 개발 워크플로우를 극대화하는 통합 기법입니다.

## 이 문서에서 다루는 내용

### 1. Spec Kit 사용법

프로젝트의 요구사항과 설계를 체계적으로 정의하는 Spec Kit의 구성과 활용법을 다룹니다.

- Spec Kit의 개념과 구조
- 작성 가이드
- 실전 활용 예시

### 2. Superpowers 스킬 사용법

AI 에이전트의 능력을 확장하는 Superpowers 스킬의 설치, 설정, 활용법을 다룹니다.

- Superpowers 스킬 개요
- 주요 스킬 소개
- 스킬 설정 및 커스터마이징

### 3. Harness 기법 적용법

Spec Kit과 Superpowers 스킬을 하나로 묶어 실전 프로젝트에 적용하는 Harness 기법을 다룹니다.

- Harness 워크플로우 개요
- 단계별 적용 가이드
- 실전 프로젝트 적용 사례
